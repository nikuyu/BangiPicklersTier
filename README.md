# ⚡ Reclub Score Converter

Convert any Reclub score sheet into a spreadsheet instantly.

---

## Setup (one time only)

1. Install Node.js from https://nodejs.org  (choose the LTS version)
2. Extract this folder somewhere on your computer

---

## How to run

### Windows
Double-click `start.bat`

### Mac / Linux
Open Terminal in this folder and run:
```
node server.js
```

Then open your browser and go to:
```
http://localhost:3000
```

---

## How to use

1. Go to your Reclub match page
2. Copy the URL (e.g. `https://api.reclub.co/export/score-sheets?m=L19INQ`)
3. Paste it into the app and click **Load Match**
4. Browse standings, per-court results, all matches
5. Click **↓ Excel** or **↓ CSV** to download

---

## Stop the server
Press `Ctrl + C` in the terminal window.
